# 문서 인덱스

이 폴더에는 Mattermost Reminder 프로젝트의 모든 문서가 포함되어 있습니다.

## 📚 문서 목록

### 개발 관련

- **[개발 계획서](DEVELOPMENT_PLAN.md)** - 프로젝트 전체 개발 계획 및 진행 상황
- **[로컬 개발 가이드](LOCAL_DEVELOPMENT.md)** - 로컬 개발 환경 설정 및 실행 방법
- **[Phase 2 설정 가이드](PHASE2_SETUP_GUIDE.md)** - Firebase 인프라 구축 가이드

### 배포 관련

- **[Vercel 배포 가이드](VERCEL_DEPLOY.md)** - 프론트엔드를 Vercel에 배포하는 방법
- **[빠른 배포 가이드](QUICK_DEPLOY.md)** - 빠른 배포를 위한 간단한 가이드
- **[Firebase Functions 배포](FIREBASE_FUNCTIONS_DEPLOY.md)** - 백엔드를 Firebase Functions에 배포하는 방법
- **[배포 옵션](DEPLOYMENT_OPTIONS.md)** - 다양한 배포 옵션 설명
- **[GitHub Actions 설정](GITHUB_ACTIONS_SETUP.md)** - CI/CD 자동화 설정 가이드

### 테스트 관련

- **[테스트 가이드](TESTING_GUIDE.md)** - 백엔드 테스트 실행 및 작성 가이드
- **[프론트엔드 테스트](FRONTEND_TESTING.md)** - 프론트엔드 테스트 환경 및 작성 가이드
- **[통합 테스트](INTEGRATION_TEST.md)** - 통합 테스트 시나리오 및 체크리스트

### 문제 해결

- **[에러 수정 내역](ERROR_FIXES.md)** - 프로젝트 개발 중 발생한 에러 및 해결 방법

## 📖 문서 읽는 순서

### 처음 시작하는 경우

1. [개발 계획서](DEVELOPMENT_PLAN.md) - 프로젝트 전체 이해
2. [로컬 개발 가이드](LOCAL_DEVELOPMENT.md) - 개발 환경 설정
3. [Phase 2 설정 가이드](PHASE2_SETUP_GUIDE.md) - Firebase 설정

### 배포하려는 경우

1. [Firebase Functions 배포](FIREBASE_FUNCTIONS_DEPLOY.md) - 백엔드 배포
2. [Vercel 배포 가이드](VERCEL_DEPLOY.md) - 프론트엔드 배포
3. [GitHub Actions 설정](GITHUB_ACTIONS_SETUP.md) - 자동화 설정

### 테스트를 작성하려는 경우

1. [테스트 가이드](TESTING_GUIDE.md) - 백엔드 테스트
2. [프론트엔드 테스트](FRONTEND_TESTING.md) - 프론트엔드 테스트
3. [통합 테스트](INTEGRATION_TEST.md) - 통합 테스트

### 문제가 발생한 경우

1. [에러 수정 내역](ERROR_FIXES.md) - 이전에 해결된 에러 확인
2. [로컬 개발 가이드](LOCAL_DEVELOPMENT.md) - 환경 설정 확인
